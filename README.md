Civilization Spatial Intelligence Mapper

This project implements a spatial intelligence system using KD-Tree to index and query civilizations based on geographic coordinates.

Features:
- KD-Tree spatial indexing
- Nearest neighbor search
- Range query search
- performance benchmarking
- CLI-based interface
- Modular architecture

Technologies:
- C++
- KD-Tree
- Spatial indexing
- Data structures and algorithms

How to run:
Compile using g++ or CMake and run executable.
